CEN Bible 성경연대기 PWA - 사사시대 추가 반영본

기준 파일:
- cen-bible-chrololgy_0518_patriarch_exodus_canaan_added.zip

추가 반영:
- 사사기의 악순환 시작
- 사사 입다
- 미가의 신상과 레위인의 타락
- 기브아의 만행
- 엘리 제사장과 사무엘의 출생

포함 파일:
- index.html
- js/app.js
- js/data.js
- js/chronology-data.js
- css/style.css
- assets/infographics/ 추가 이미지
- assets/maps/ 추가 지도·시각자료 이미지
- manifest.json
- sw.js
- icons/

업로드 방법:
1. 압축을 풉니다.
2. GitHub의 cen-bible-chrololgy 저장소 루트에 전체 덮어쓰기 업로드합니다.
3. Commit changes를 누릅니다.
4. GitHub Pages 배포 후 새로고침 또는 캐시 삭제 후 확인합니다.

확인 경로:
시대별 연대기 보기 → 사사 시대 → 추가 핵심사건 클릭 → 상세보기
