# CEN Bible 성경연대기 독립 PWA v8

## 구조
- `index.html`: 화면 뼈대
- `css/style.css`: CEN Bible 톤 UI
- `js/chronology-data.js`: 시대/사건 데이터
- `js/app.js`: 화면 동작
- `assets/infographics/`: 원본 인포그래픽
- `sw.js`: PWA 캐시

## 현재 완성된 상세 탭카드
- 창조
- 아담과 하와
- 에덴동산 추방
- 가인과 아벨
- 노아의 방주
- 바벨탑

나머지 시대는 사건칩과 미리보기 구조만 잡혀 있으며, 인포그래픽을 추가하면 같은 방식으로 확장할 수 있습니다.
