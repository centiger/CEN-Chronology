v86 연결탐험 N개 버튼 활성화 수정

수정 내용:
1. 연결탐험 N개 버튼 클릭 이벤트를 document click handler 최상단에 추가
2. 기존 코드가 존재하지 않는 renderHub/go('hub')를 호출하던 문제 수정
3. 현재 앱의 실제 허브 표시 함수인 renderHubOverlay()를 사용하도록 변경
4. 허브 1개는 바로 오버레이 열기
5. 허브 2개 이상은 선택창 열고 선택한 허브 오버레이 표시

확인:
- 십자가 같은 다중 허브 사건에서 '연결탐험 N개' 클릭
- 선택창이 뜨고 허브 선택 시 오버레이가 열려야 정상
