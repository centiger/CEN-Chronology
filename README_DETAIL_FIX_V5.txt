   - 지도 crop 이미지가 있으면 지도 crop 표시
   - crop 이미지가 없으면 원본 인포그래픽으로 대체 표시
   - 크게 보기 버튼도 동일한 보강 경로 사용

3. 원본 인포그래픽 보기 연결 보강
   - ORIGINAL_INFOGRAPHICS → EVENTS.image → EVENTS.originalImage 순서로 경로 탐색
   - 누락 시 사용자 알림 표시
