1. 전체 핵심사건 상세화면의 6.핵심사건 렌더링 규칙 수정
2. [제목, 설명, 성경구절] 구조일 때:

3. 설명문 끝에 붙은 성경구절도 가능한 경우 자동 분리
   예: '구원의 약속을 주심 (창 3:15)' → 설명 / 창 3:15
4. 일곱 인 하드픽스 유지
